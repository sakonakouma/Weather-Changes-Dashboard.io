The purpose of this project was to analyze how the weather changes
as you get closer to the equator. To accomplish this analysis, the data
were pulled from the OpenWeatherMap API to assemble a dataset
over 500 cities.
Python and Matplotlib libraries were used to explore the dataset
Local weather caracteristics were analyzed.
